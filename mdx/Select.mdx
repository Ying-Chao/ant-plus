# Select 选择器

## 使用

```js
import { Select } from 'antx';
```

## 示例

<Playground>
  <Select
    data={[
      { id: 'tiger', name: '🐅 小脑斧' },
      { id: 'dog', name: '🐶 小狗子' },
      { id: 'cat', name: '🐱 小猫崽' },
    ]}
    keys={['id', 'name']}
    search
    clear
    msg="选个宠物吧"
  />
</Playground>
