# Cascader 级联选择

## 使用

```js
import { Cascader } from 'antx';
```

## 示例

<Playground>
  <Cascader
    data={[
      {
        id: 'tiger',
        name: '🐅 小脑斧',
        children: [{ id: 'little tiger', name: '🐅 小脑斧的儿子' }],
      },
      { id: 'dog', name: '🐶 小狗子', children: [{ id: 'little dog', name: '🐶 小狗子的儿子' }] },
      { id: 'cat', name: '🐱 小猫崽' },
    ]}
    keys={['id', 'name']}
    onChange={(value, selectedOptions) => {
      console.log('value: ', value);
      console.log('selectedOptions: ', selectedOptions);
    }}
    search
    clear
    msg="找一找"
  />
</Playground>
